# 贡献指南

感谢你对这个实验性质项目的兴趣！

## 这是一个个人探索项目

- 代码可能有不完善的地方
- 文档可能不及时
- 欢迎提出 Issue 和 PR

## 如何贡献

1. **报告 Bug**: 提一个 Issue，说明复现步骤
2. **建议功能**: 提一个 Issue，描述你的想法
3. **提交代码**:
   - Fork 本项目
   - 创建分支: `git checkout -b feature/xxx`
   - 提交: `git commit -m "feat: xxx"`
   - 推送: `git push origin feature/xxx`
   - 提 PR

## 代码风格

- Python: `black . && isort .`
- TypeScript: `npm run lint`

## 保持耐心

这是一个个人项目，回复可能不及时，感谢理解！
